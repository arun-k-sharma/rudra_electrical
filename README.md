# ⚡ Rudra Electricals

A modern Electrical Shop Website built using **Node.js, Express.js, MongoDB, EJS, and Cloudinary**. The website allows customers to browse electrical products and enables the shop owner to manage products through an admin dashboard.

---

## 🚀 Features

### Customer Side
- 🏠 Modern Landing Page
- 📦 Browse Products
- 🔍 Product Details Page
- 🛠️ Services Page
- 🖼️ Gallery
- 📞 Contact Page
- 📱 Fully Responsive Design

### Admin Panel
- 🔐 Secure Admin Login
- ➕ Add Products
- ✏️ Edit Products
- ❌ Delete Products
- ☁️ Image Upload using Cloudinary
- 📊 Dashboard

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/arun-k-sharma/rudra-electricals.git
```

Go inside the project

```bash
cd rudra-electricals
```

Install dependencies

```bash
npm install
```

Create a `.env` file

```env
PORT=3000

MONGODB_URI=your_mongodb_connection_string

JWT_SECRET=your_secret_key

CLOUDINARY_CLOUD_NAME=your_cloud_name
CLOUDINARY_API_KEY=your_api_key
CLOUDINARY_API_SECRET=your_api_secret
```

Start the server

```bash
npm start
```

For development

```bash
npm run dev
```
